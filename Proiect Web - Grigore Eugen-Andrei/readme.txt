Proiectul l-am facut cu ajutorul npm. 
Este necesara instalarea npm, dupa care trebuie rulat npm install in folderul cu proiectul.
Pentru a rula aplicatia trebuie sa se ruleze comanda npm run start intr-o consola
si trebuie rulat si server-ul php in alta consola cu comanda php -S localhost:3000.

Ca si mentiune partea de stergere a elementelor todo nu functioneaza, am incercat cateva ore bune
sa rezolv situatia dar nu a mers.
